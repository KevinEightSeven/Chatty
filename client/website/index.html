// features

Everything you need.
Nothing you don't.

Built for power users and streamers who want full control over their Twitch chat experience.

▣

Multi-Pane Splits

Watch multiple chats at once with a flexible drag-and-drop split system. Horizontal, vertical, and nested column layouts.

✨

Full Emote Support

Twitch, BTTV, FrankerFaceZ, and 7TV emotes render inline with full badge support including global and channel-specific badges.

🛡

Streamer Tools

Built-in alert system with OBS overlay support. Configure followers, subs, bits, and raid alerts with custom images, sounds, and variants.

🎬

Trigger Board

Create custom sound and video triggers linked to channel point redemptions. Auto-creates Twitch rewards with a dedicated OBS Browser Source.

🔔

Activity Feed & Mod Actions

Embedded Twitch Activity Feed and Mod Actions panels showing follows, subs, raids, bans, and timeouts in real time. Persistent login.

👥

User List & Profiles

See who's in chat with role-based categories and join/leave notifications. Click any username for a floating profile card with mod actions.

💬

Whispers

Dedicated whisper window intercepts all private messages. Conversation tabs, unread badges, and instant reply support.

❤

Room Mode Indicators

Live icons show followers-only, sub-only, and emote-only modes right in the channel header. Updates instantly.

⌘

Slash Commands

Full Twitch command support with autocomplete. /ban, /timeout, /mod, /vip, /raid, /announce, and more.

🔄

Auto-Updater

Checks for new releases on startup. One-click download and install for seamless updates across Linux, Windows, and AUR.
